Ionic アプリは、コンポーネントと呼ばれる高レベルのビルディングブロックで構成されており、アプリの UI を迅速に構築できます。Ionic には、カード、リスト、タブなど、多数のコンポーネントが標準装備されています。基本を理解したら、各コンポーネントとサブコンポーネントの完全なリストについては[API Index](api.md)を参照してください。
