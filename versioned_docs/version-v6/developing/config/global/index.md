The `setupIonicReact` function must be called before rendering any Ionic components (including `IonApp`).

```tsx title="App.tsx"
import { setupIonicReact } from '@ionic/react';

setupIonicReact({
  rippleEffect: false,
  mode: 'md',
});
```
