### download 

| | |
| --- | --- |
| **Description** | This attribute instructs browsers to download a URL instead of navigating to it, so the user will be prompted to save it as a local file. | **Attribute** | `download` |
| **Type** | `string ｜ undefined` |
| **Default** | `undefined` |
