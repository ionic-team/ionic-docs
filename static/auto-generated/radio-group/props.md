

### allowEmptySelection 

| | |
| --- | --- |
| **Description** | If `true`, the radios can be deselected.<br /><br />自動翻訳: `true`の場合、ラジオを非選択にすることができる。 |

| **Type** | `boolean` |
| **Default** | `false` |
